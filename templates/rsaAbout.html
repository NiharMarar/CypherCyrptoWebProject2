What is RSA

RSA uses two public keys, and one private key to encrypt a message. Each character in the message is turned into a number, and run through a modulo equation. This process encrypts or decrypts the equation. The keys used in the formula determine whether the message is encrypted, or decrypted. If both public keys are used, the message is encrypted. If the private key and first public key are used, the message is decrypted. Not any number works as keys, the keys need to be generated in order for the encryption to work. If the wrong keys are used, message will not be able to be reverted.

Key Generation

In order to encrypt a message, the user input must be turned into numbers. On this website, the characters are converted using the ascii library. When converted, letters are run through the equation messagekey 1 mod key 2. "mod" takes the remainder of the first value divided by the second value. If two characters have the same value after mod, the key is invalid. The public and private keys undo each other when run through the equation. In order to find keys that work they must be generated.
