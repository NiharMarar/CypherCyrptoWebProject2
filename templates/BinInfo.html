Binary is a basic, base-2 number system that was invented by Gottfried Leibniz around 1679. The only two numbers that binary is made of are 0's and 1's. These numbers are meant to represent off and on reactions; 0 means no flow of electricity in a circuit (or off), and 1 means electricity being flowed (or on). This number system is what makes up much of the date we use today, such as the date necessary for computer processors. The reason why computers use binary is because it is simple to read, easy to detect, positive and negative poles of magnetic media are quickly translated into binary code, and it is the most efficient way to control logic circuits. Binary codes are in 8-digits segments, and to read them, you start from the right-end. The 1's and )'s are converted into the number 2 to the power of 0-7, starting from right to left. So the last digit would be 2^0, the next would be 2^1, the next is 2^2 and so on. If the digit is a 0, then the result is 0 because it would be "off," therefore, having no outcome. An example would be:

1 - 0 - 1 - 1 - 0 - 0 - 1 - 0

2^7 - 2^6 - 2^5 - 2^4 - 2^3 - 2^2 - 2^1 - 2^0
